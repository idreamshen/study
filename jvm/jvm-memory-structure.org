* 堆（Heap）
由新生代 Young Generation 和老年代 Old Generation 组成。
- 新生代：Eden 伊甸区、From Survivor 幸存者区、To Survivor 幸存者区
- 老年代：Tenured

* 方法区（Method Area）
用于存储已被虚拟机加载的类信息、常量、静态变量、即编译器编译后的代码
